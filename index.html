Naming Conventions:

Always lowercase

  • camelCase
  • lowercase
  • MiXeDc AsE
  • Capital Letter
  • UPPERCASE
  • Sentence case
  • This is a Title Case

Good examples:
tree-branch-logo
main.css

Bad examples:
Today-In-History.html
meatEatingDinosaur.jpg

Only letters, numbers and dashes (-)

Commit message best practises

  • Minimum 3 words & 10 characters
  • Proper spelling & grammar
  • Capital sarting letter, no period
  • Must start with an imperative verb: Create, Fix, Add, Solve, etc.
  • Pretend every commit starts with the phrase: “This commit will…”

Syntax

Headings:
< h1 > open tag
< /h1 > close tag

< !DOCTYPE html >
The HTML document type declaration, also known as DOCTYPE , is the first line of code required in every HTML or XHTML document.
The DOCTYPE declaration is an instruction to the web browser about what version of HTML the page is written in.

Link Text

The opening <a> tells where the link should start and the closing </a> indicates where the link ends. To create a hyperlink, you should use the <a> tag and href attribute, the value of which is the URL, or location, where the link is pointing to.

<a href="link text" =""description"</a>

Link Image

Images in websites are not embedded in the website but linked to by the HTML document. Images need two attributes to specify extra details about the image: the location of the image and the alternative content.

<img src="image link" alt="image description">
